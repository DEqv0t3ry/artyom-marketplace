# Запуск проекта

## Завсисимости:
* PHP 8.3
* Composer
* Docker
## 1. Установка
* Склонировать git-репозиторий
* Выполнить команду ```composer install```
* Для запуска проекта выполняем команду ```./vendor/bin/sail up -d```
* Для запуска миграций выполняем команду ```./vendor/bin/sail artisan migrate --seed```
* Открываем 0.0.0.0
